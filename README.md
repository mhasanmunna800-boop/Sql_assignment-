## Key Concepts Covered

- **SELECT, WHERE, ORDER BY** - Basic query structure and filtering
- **JOINs (INNER, LEFT, RIGHT)** - Combining tables
- **GROUP BY, HAVING** - Aggregation and filtering groups
- **Subqueries** - Nested queries for complex logic
- **String Functions** - CONCAT, LEFT, SUBSTRING, UPPER, LOWER, LIKE
- **Aggregate Functions** - COUNT, SUM, MAX, MIN, AVG, GROUP_CONCAT
- **Date Operations** - BETWEEN for date range filtering
